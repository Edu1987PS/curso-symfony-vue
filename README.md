# VB→PHP: Curso de Symfony + Vue.js

Curso interactivo para programadores VB.NET que quieren transit a PHP, Symfony y Vue.js.

## Contenido

23 lecciones cubriendo:

- **Modulo 1: Entorno** - XAMPP, Virtual Hosts, Composer
- **Modulo 2: PHP Fundamentos** - Variables, tipos, operadores, condicionales, bucles, funciones
- **Modulo 3: POO en PHP** - Clases, herencia, interfaces, traits, autoloading
- **Modulo 4: Symfony** - Rutas, controladores, Twig, Doctrine, formularios
- **Modulo 5: Vue.js** - Composition API, directivas, componentes
- **Modulo 6: Proyecto Final** - API REST + Vue

## Tecnologias

- HTML5 + CSS3 (dark theme estilo VS Code)
- Vue 3 (CDN, Composition API)
- Prism.js (syntax highlighting)
- localStorage (persistencia de progreso)
- GitHub Pages (hosting)

## Instalacion local

1. Clona el repositorio
2. Abre `index.html` en un navegador
3. Login: `edu` / `curso2026`

## Despliegue en GitHub Pages

1. Sube el proyecto a un repositorio GitHub
2. Ve a Settings > Pages
3. Selecciona la rama `main` y guarda
4. Esperaa unos minutos y accede a `https://tu-usuario.github.io/curso-symfony-vue/`
